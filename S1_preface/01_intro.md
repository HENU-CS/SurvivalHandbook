# 前言

由于河南大学是很好的学校，但是：

1. **学生与发达地区的家庭之间的信息差实在太大：** 学生主要来源于较为落后的地区，父母们的思维和北京/上海/福建/广州等地区有非常大的间隔。这种间隔，甚至是几十年几百年都难以抹平的东西。（我起初想说信息差，但是学校或者网络其实并不是主要造成的原因。家庭才是。）
2. **学生由于信息差，白白浪费时间，空消磨才华：** 以至于本身并不弱于985中流学生，但是他们在前进的时候，我们在沉沦。
3. **规则导向上，不够先进：** 并不是说好学校的规定就有很好。哪怕是复旦或者北大的这些地方，其实他们都非常的陈旧，在制度上没有办法完全迸发出学生的活力。比如复旦或者北大，他们一个gpa制度比较低下，一个对本科生培养力度不够。所以弱于交大（复旦培养出的AP级别的学生仅仅是交大的1/3）和清华（北大图灵班的学生总体实力微弱于清华姚班）。
4. **对前沿抓的不紧：** 一部分原因是老师不够前沿：这种是由于奖励制度不够前沿. 一部分是由于我们河南吸引不够非常强的人才：就算是校友出去之后，除非河南大学能提供比沿海的薪资强1.5倍甚至两倍，否则也是吸引不来的。其实内陆的学校都有这个问题。假如说河南大学在深圳办的研究院比较好的话，也许会有一点吸引性。但是恐怕不会吸引到比较强的博士来担任AP. 他们大体上有更好的选择。

所以总结来说。原因是多方面的，你不能说是学校不够努力，你也不能说是这个制度上有问题。因为很多地方都有这个问题。而且这个问题也不是很容易就解决的。

这个问题恐怕在几十年内也不好解决，只能依靠自然经济增长以及人口下降来变的更好一点。考虑到文化的氛围，内陆的学校总归是吃亏的。因此，河南大学长期仍然将会是一个普惠性的学校，暂时没有办法，成为很强的研究型学校。我们应该知道这个客观事实。

**但是对于学生来说：** 你完全可以自发向上，自强不息。通过努力解决自己后天的不足，然后成为顶级的学生。理论上，河南大学的学生完全能够够得上强985以外的所有上线，只要你能解决你的信息差以及自发的进步。

这本生存手册就是帮助你能在河南大学成为全国最强的本科生之一。然后其中的内容基本上仅用于CS学生。目前其他专业的很难改命，我们暂时不做讨论。
